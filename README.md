# zxy.vc

Landing de **ZXY Ventures** — venture crafting.

> We build, operate, and create value with AI.

Sitio estático de una sola página, servido con GitHub Pages en [zxy.vc](https://zxy.vc).

## Estructura

- `index.html` — el landing completo: HTML/CSS/JS vanilla autocontenido, sin build ni dependencias locales (solo Google Fonts). Los tokens de marca van embebidos en su `<style>`.
- `404.html` — página de error con el mismo lenguaje visual.
- `og.png` — imagen social 1200×630 (wordmark sobre `--bg`, placa `zxy.vc`).
- `CNAME`, `robots.txt`, `sitemap.xml` — configuración de dominio e indexación.

## Marca

- Tipografía: Geist / Geist Mono (Google Fonts, pesos 300/400/500).
- Modo claro por default; el toggle `dark`/`light` persiste en `localStorage` (`zxy-theme`).
- ZXY vive en grises; el único color es el dot de cada venture.
